## DEV SETUP

### install venv

`python -m pip install venv`


### run venv

`python -m venv venv`

### activate venv

#### on linux

`source venv/bin/activate`

#### on windows

`source venv/Scripts/activate`

### install packages

`python -m pip install -r requirements.txt`
